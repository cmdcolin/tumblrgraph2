Instructions: copy and paste the notes from a given post on tumblr (basically, go the the actual URL for the post and copy them) and then use the custom settings here to customize the layout. Some graph layouts require a "simulation" phase so the layout time parameter is configurable, during which time the text will say "Optimizing layout". You can interact with the nodes before and after this phase to affect the overall layout.
